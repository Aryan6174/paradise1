# PowerFit Gym Website - MERN Stack

A complete gym website built with the MERN (MongoDB, Express.js, React.js, Node.js) stack.

## Features

- 🏋️ **Home Page** - Hero section, features, classes preview, trainers, and membership plans
- 📖 **About Page** - Gym story, mission, vision, and values
- 🎯 **Classes Page** - Browse and filter fitness classes, view schedules
- 👨‍🏫 **Trainers Page** - Meet the expert trainers
- 💳 **Membership Page** - View and subscribe to membership plans
- 📞 **Contact Page** - Contact form and gym information
- 🔐 **Authentication** - User registration and login
- 📊 **Dashboard** - User profile, enrolled classes, and membership status

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- bcrypt.js for password hashing

### Frontend
- React.js
- React Router DOM
- React Context API for state management
- Axios for API calls
- React Icons
- React Toastify for notifications
